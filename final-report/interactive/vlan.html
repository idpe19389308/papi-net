Σχέδιο VLAN

Επιλέξτε ένα υπο-υποδίκτυο

Σχέδιο VLAN: κάθε LAN /24 διαιρείται σε 3 χρησιμοποιήσιμα /26 (VLAN 10/20/30) + 1 εφεδρικό. Κάντε κλικ σε κάθε υπο-υποδίκτυο για IP plan, switch ports, και subinterface config.

Κανόνας μάσκας /26 (255.255.255.192): 64 διευθύνσεις ανά μπλοκ, 62 χρησιμοποιήσιμα hosts (μείον network & broadcast). Το gateway είναι πάντα η πρώτη χρήσιμη (.1, .65, .129).
