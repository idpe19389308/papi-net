Στιγμιότυπα οθόνης από το Cisco Packet Tracer
=============================================

Αρχεία:
  01-topology.png              — Πλήρης τοπολογία στο PT workspace με ονόματα συσκευών
  02-show-ip-int-brief.png     — R-IOA: show ip interface brief (όλα τα subinterfaces up)
  03-show-ospf-neighbor.png    — R-THE: show ip ospf neighbor (γείτονες FULL)
  04-show-ip-route-ospf.png    — R-CHA: show ip route ospf (LAN /26 ως O)
  05-show-vlan-brief.png       — SW-ATH: show vlan brief (VLAN 10/20/30 + ports)
  06-show-interfaces-trunk.png — SW-PAT: show interfaces trunk (Fa0/24 allowed 10,20,30)
  07-ping-success-intra-vlan.png — PC VLAN10 Αθήνας → PC VLAN10 Χανίων (επιτυχία)
  08-ping-fail-inter-vlan.png    — PC VLAN10 Αθήνας → PC VLAN20 Αθήνας (αποτυχία / ACL block)
  09-ping-server.png             — Ping προς Web server Ιωαννίνων από PC
